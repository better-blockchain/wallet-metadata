# wallet-metadata
crypto wallet metadata


## feature:

- generate crypto wallet token contract metadata:
    - eth
    - bsc
- source: `data`    
- dist: `dist`


## quickstart:


- install:

```bash
make install
```


- run:


```bash

make run 

# or

task run

```

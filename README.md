# wallet-metadata
crypto wallet metadata


## feature:

- generate crypto wallet token contract metadata:
    - eth
    - bsc
- source: `data`    
- dist: `dist`


## quickstart:


- install:

```bash
make install
```

- init:

```bash
make init 

# or:

task init

```


- run:


```bash
make gen 

# or

task gen

```
